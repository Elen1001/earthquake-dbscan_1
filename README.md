# Երկրաշարժերի տվյալների կլաստերավորում DBSCAN-ի միջոցով

Այս նախագիծն իրականացնում է երկրաշարժերի համաշխարհային տվյալների վերլուծություն և խտության վրա հիմնված կլաստերավորում (Density-Based Clustering)՝ օգտագործելով **DBSCAN** ալգորիթմը: Նպատակն է հայտնաբերել սեյսմիկ ակտիվության բարձր խտությամբ տարածքները և մեկուսացնել անոմալիաները (աղմուկը):

## 📌 Նախագծի համառոտագիր (Overview)
Աշխարհագրական տվյալների կլաստերավորման ժամանակ ավանդական մեթոդները (օրինակ՝ K-Means-ը) հաճախ ձախողվում են, քանի որ դրանք ենթադրում են կլաստերների կլորավուն կամ սֆերիկ ձև: Երկրաշարժերի օջախները ձևավորում են անկանոն երկրաչափական պատկերներ: **DBSCAN (Density-Based Spatial Clustering of Applications with Noise)** ալգորիթմը թույլ է տալիս՝
* Հայտնաբերել կամայական ձև ունեցող կլաստերներ:
* Ավտոմատ կերպով որոշել կլաստերների քանակը:
* Նույնականացնել և զտել աղմուկը (noise/outliers)՝ առանձնացնելով եզակի երկրաշարժերը հաճախակի կրկնվողներից:

## 📊 Տվյալների բազայի կառուցվածք (Dataset Overview)
Նախագծում օգտագործվում է պատմական երկրաշարժերի տվյալների բազա, որը պարունակում է **23,412 տող** և **21 սյունակ**:

### Հիմնական հատկանիշները (Features):
* **Աշխարհագրական դիրք:** `Latitude` (Լայնություն), `Longitude` (Երկայնություն)
* **Սեյսմիկ ցուցանիշներ:** `Depth` (Խորություն), `Magnitude` (Մագնիտուդ), `Magnitude Type`
* **Ժամանակային տվյալներ:** `Date`, `Time`
* **Մետատվյալներ և Սխալներ:** `ID`, `Root Mean Square`, `Depth Error`, `Magnitude Error`

* Օգտագործելով `zipfile` մոդուլը, `database.csv` ֆայլը ավտոմատ կերպով արտահանվում է արխիվից:
* Տվյալների առաջնային զննում `.shape`, `.info()` և նկարագրական վիճակագրության `.describe()` ֆունկցիաների միջոցով:

### 2. Տվյալների Մշակում (Data Preprocessing & Cleaning)
* **Բացակայող արժեքների (Null Values) վերլուծություն:** Նախագծում հաշվարկվում են սյունակների դատարկ արժեքների շեմերը: Այնպիսի սյունակներ, ինչպիսիք են `Depth Error`-ը կամ `Magnitude Error`-ը, ունեն բացակայող տվյալներ, որոնք համապատասխանաբար մշակվում կամ զտվում են:
* **Հատկանիշների ընտրություն (Feature Selection):** Կլաստերավորման համար առանձնացվում են աշխարհագրական կոորդինատները (`Latitude`, `Longitude`) և սեյսմիկ բնութագրիչները (`Depth`, `Magnitude`):
* **Մասշտաբավորում (Feature Scaling):** Քանի որ երկայնությունը/լայնությունը և խորությունը ունեն չափման տարբեր միավորներ, կիրառվում է `StandardScaler`: Սա կանխում է մոդելի կողմից որևէ հատկանիշի արհեստական գերակշռումը էպսիլոն ($\epsilon$) հեռավորությունը հաշվարկելիս:
